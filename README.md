# Address Book

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```


### Functionalities

- Create, Read, Update and Delete a user

### User fields

- First Name
- Last Name
- Email
- Phone Number
- Notes
- Date of birth

### Validation

- Validates all user input field

### Testing

```
npm run test
```
